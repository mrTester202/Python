### 3.1 –°–ø–∏—Å–∫–∏
–°–ø–∏—Å–∫–∏ –≤ Python - —ç—Ç–æ —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã–µ –∏–∑–º–µ–Ω—è–µ–º—ã–µ –∫–æ–ª–ª–µ–∫—Ü–∏–∏ –æ–±—ä–µ–∫—Ç–æ–≤ –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã—Ö —Ç–∏–ø–æ–≤ (–æ–Ω–∏ –º–æ–≥—É—Ç –≤–∫–ª—é—á–∞—Ç—å –ª—é–±–æ–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö: —Ü–µ–ª—ã–µ —á–∏—Å–ª–∞, —Å—Ç—Ä–æ–∫–∏, –¥—Ä—É–≥–∏–µ —Å–ø–∏—Å–∫–∏ –∏ —Ç.–¥.). –ö —ç–ª–µ–º–µ–Ω—Ç–∞–º —Å–ø–∏—Å–∫–∞ –º–æ–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å –¥–æ—Å—Ç—É–ø, –∏—Å–ø–æ–ª—å–∑—É—è –∏–Ω–¥–µ–∫—Å—ã. 

–ü—Ä–∏–º–µ—Ä: 
```python

print(my_list[1])  # –í—ã–≤–µ–¥–µ—Ç: Hello
my_list[1] = "World"  # –ò–∑–º–µ–Ω–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å–ø–∏—Å–∫–∞

```

**–û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å–ø–∏—Å–∫–∞–º–∏:**
1. list.append(x) –î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –≤ –∫–æ–Ω–µ—Ü —Å–ø–∏—Å–∫–∞.
2. list.extend(iterable) –†–∞—Å—à–∏—Ä—è–µ—Ç —Å–ø–∏—Å–æ–∫, –¥–æ–±–∞–≤–ª—è—è –≤ –∫–æ–Ω–µ—Ü –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ.
3. list.insert(i, x) –í—Å—Ç–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –Ω–∞ –∑–∞–¥–∞–Ω–Ω—É—é –ø–æ–∑–∏—Ü–∏—é.
4. list.remove(x) –£–¥–∞–ª—è–µ—Ç –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ —Å–ø–∏—Å–∫–µ, –∫–æ—Ç–æ—Ä—ã–π —Ä–∞–≤–µ–Ω x.
5. list.pop([i]) –£–¥–∞–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –Ω–∞ –∑–∞–¥–∞–Ω–Ω–æ–π –ø–æ–∑–∏—Ü–∏–∏ –≤ —Å–ø–∏—Å–∫–µ –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ–≥–æ.
6. list.clear() –£–¥–∞–ª—è–µ—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ —Å–ø–∏—Å–∫–∞.
7. list.index(x[, start[, end]]) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º x.
8. list.count(x) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑, –∫–æ—Ç–æ—Ä–æ–µ x –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è –≤ —Å–ø–∏—Å–∫–µ.
9. list.sort(key=None, reverse=False) –°–æ—Ä—Ç–∏—Ä—É–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã —Å–ø–∏—Å–∫–∞ –Ω–∞ –º–µ—Å—Ç–µ.
10. list.reverse() –†–∞–∑–≤–æ—Ä–∞—á–∏–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã —Å–ø–∏—Å–∫–∞ –Ω–∞ –º–µ—Å—Ç–µ.

### 3.2 –ö–æ—Ä—Ç–µ–∂–∏

**–ö–æ—Ä—Ç–µ–∂–∏** –æ—á–µ–Ω—å –ø–æ—Ö–æ–∂–∏ –Ω–∞ —Å–ø–∏—Å–∫–∏, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º —Ç–æ–≥–æ, —á—Ç–æ –æ–Ω–∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã (–∏—Ö –Ω–µ–ª—å–∑—è –∏–∑–º–µ–Ω–∏—Ç—å –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è). –ò—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –∫–æ—Ä—Ç–µ–∂–∏ –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –æ–¥–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π.

–ü—Ä–∏–º–µ—Ä:
```python

print(my_tuple[1])  # –í—ã–≤–µ–¥–µ—Ç: Hello
```

–ö–æ—Ä—Ç–µ–∂–∏ –Ω–µ –∏–º–µ—é—Ç –º–µ—Ç–æ–¥–æ–≤ –¥–ª—è –∏–∑–º–µ–Ω–µ–Ω–∏—è —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ –∏–∑-–∑–∞ –∏—Ö –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–æ—Å—Ç–∏, –Ω–æ –≤—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–±—â–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏, —Ç–∞–∫–∏–µ –∫–∞–∫ **len**(tuple) –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –¥–ª–∏–Ω—ã –∫–æ—Ä—Ç–µ–∂–∞ –∏ tuple.**count**(x) –¥–ª—è –ø–æ–¥—Å—á–µ—Ç–∞ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ —Ä–∞–∑, –∫–æ—Ç–æ—Ä–æ–µ x –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è –≤ –∫–æ—Ä—Ç–µ–∂–µ.

### 3.3 –ú–Ω–æ–∂–µ—Å—Ç–≤–∞

**–ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –≤ Python** - —ç—Ç–æ –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã–µ –∫–æ–ª–ª–µ–∫—Ü–∏–∏ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –û–Ω–∏ –ø–æ–ª–µ–∑–Ω—ã, –∫–æ–≥–¥–∞ –≤–∞–º –Ω—É–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç—å —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–æ–ª–ª–µ–∫—Ü–∏–∏ –∏–ª–∏ –µ—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–ª—É—á–∏—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ –∫–æ–ª–ª–µ–∫—Ü–∏–∏.

–ü—Ä–∏–º–µ—Ä:
```python
my_set = {1, 2, 3, 4, 4, 4}
print(my_set)  # –í—ã–≤–µ–¥–µ—Ç: {1, 2, 3, 4} - –¥—É–±–ª–∏–∫–∞—Ç—ã –±—ã–ª–∏ —É–¥–∞–ª–µ–Ω—ã
```

**–ú–µ—Ç–æ–¥—ã –º–Ω–æ–∂–µ—Å—Ç–≤–∞:**
1. set.add(x) –î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ.
2. set.remove(x) –£–¥–∞–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞. –ü–æ–¥–Ω–∏–º–∞–µ—Ç KeyError, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç –Ω–µ –Ω–∞–π–¥–µ–Ω.
3. set.discard(x) –£–¥–∞–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç, –µ—Å–ª–∏ –æ–Ω –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ.
4. set.pop() –£–¥–∞–ª—è–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞.
5. set.clear() –£–¥–∞–ª—è–µ—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞.
6. set.union(*others) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å–æ –≤—Å–µ–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –∏–∑ –≤—Å–µ—Ö –º–Ω–æ–∂–µ—Å—Ç–≤.
7. set.intersection(*others) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏, –æ–±—â–∏–º–∏ –¥–ª—è –≤—Å–µ—Ö –º–Ω–æ–∂–µ—Å—Ç–≤.
8. set.difference(*others) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –≤—Ö–æ–¥—è—Ç –≤ –¥—Ä—É–≥–∏–µ.

### 3.4 –°–ª–æ–≤–∞—Ä–∏

**–°–ª–æ–≤–∞—Ä–∏ –≤ Python** - —ç—Ç–æ –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç –ø–∞—Ä—ã –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ. –ö–ª—é—á–∏ –≤ —Å–ª–æ–≤–∞—Ä–µ —É–Ω–∏–∫–∞–ª—å–Ω—ã –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã, –∑–Ω–∞—á–µ–Ω–∏—è –º–æ–≥—É—Ç –±—ã—Ç—å –ª—é–±–æ–≥–æ —Ç–∏–ø–∞.

–ü—Ä–∏–º–µ—Ä:
```python
my_dict = {'name': 'John', 'age': 27}
print(my_dict['name'])  # –í—ã–≤–µ–¥–µ—Ç: John
my_dict['age'] = 28  # –ò–∑–º–µ–Ω–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –ø–æ –∫–ª—é—á—É

```

**–û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å–ª–æ–≤–∞—Ä—è–º–∏:**
1. dict.get(key[, default]) –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –∫–ª—é—á–∞, –µ—Å–ª–∏ –æ–Ω–æ –µ—Å—Ç—å –≤ —Å–ª–æ–≤–∞—Ä–µ.
2. dict.keys() –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∫–ª—é—á–µ–π —Å–ª–æ–≤–∞—Ä—è.
3. dict.values() –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π —Å–ª–æ–≤–∞—Ä—è.
4. dict.items() –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å–ª–æ–≤–∞—Ä—è.
5. dict.update([other]) –û–±–Ω–æ–≤–ª—è–µ—Ç —Å–ª–æ–≤–∞—Ä—å –¥—Ä—É–≥–∏–º —Å–ª–æ–≤–∞—Ä–µ–º –∏–ª–∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–º.
6. dict.clear() –£–¥–∞–ª—è–µ—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ —Å–ª–æ–≤–∞—Ä—è.
7. dict.pop(key[, default]) –£–¥–∞–ª—è–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ —Å–ª–æ–≤–∞—Ä—è —Å –¥–∞–Ω–Ω—ã–º –∫–ª—é—á–æ–º.
8. dict.popitem() –£–¥–∞–ª—è–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–∞—Ä—É (–∫–ª—é—á, –∑–Ω–∞—á–µ–Ω–∏–µ) –∏–∑ —Å–ª–æ–≤–∞—Ä—è.
